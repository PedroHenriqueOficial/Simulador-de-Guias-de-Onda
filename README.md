# Simulador de Guias de Onda

Simulador de guia de onda WR-10  baseado no método das diferenças finitas no domínio do tempo (FDTD).

Guia de Onda Retangular Comercial escolhido para este trabalho :
    
    • WR-10

    ↪ Utilização :
     
        ↳ Para ondas milimétricas - Banda W - comum em radares da alta resolução e imagens de segurança.

    ↪ Aplicação :
     
        ↳ Telecomunicações e 5G (mmWave)

    ↪ Especificações :

        ↪ Faixa de Operação (frequência) :
        
            ↳ Faixa de Frequência Recomendada                 -     75 GHz a 110 GHz
            ↳ Frequência de corte (modo fundamental - TE10)   -     59,015 GHz
            ↳ Frequência de corte (modo superior)             -     118,03 GHz

        ↪ Dimensões Físicas (internas) :

            ↳ Largura (dimensão 'a')                          -     2,54 mm
            ↳ Altura (dimensão 'b')                           -     1,27 mm

        ↪ Material :

            ↳ Cobre 
            ↳ Latão
            ↳ Alumínio

    ↪ Considerações :
            
        ↳ Paredes metálicas perfeitamente condutoras (PEC)
        ↳ Interior preenchidos por ar
        ↳ Permeabilidade magnética (μ)                       ≈      4π x 10⁻⁷ H/m 
        ↳ Permeabilidade elétrica (ε)                        ≈      8,854 x 10⁻¹² F/m
